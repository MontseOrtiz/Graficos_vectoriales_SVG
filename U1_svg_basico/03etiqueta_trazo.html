Etiqueta de trazo (path)

La etiqueta "path" se utiliza para definir un camino. Un camino puede ser una línea recta, una curva, una forma compleja, o incluso el contorno de un objeto.

Atributos de movimiento más comunes:

d (data): Este atributo define los comandos del camino. Es una cadena de texto que especifica los movimientos y las curvas del camino.

M (moveto): Este comando se utiliza para mover el lápiz (o punto de inicio) a una posición específica sin dibujar nada. Por ejemplo, M x y moverá el lápiz a las coordenadas (x, y).

Z (closepath): Este comando cierra el camino actual volviendo al punto de inicio del camino, creando una forma cerrada. No requiere argumentos.

H: Dibuja una línea horizontal desde la posición actual del lápiz hasta un valor de coordenada x específico. Por ejemplo, H x dibujará una línea horizontal hasta la coordenada x.

L (lineto): Este comando dibuja una línea recta desde la posición actual del lápiz hasta un punto específico. Por ejemplo, L x y dibujará una línea recta hasta las coordenadas (x, y).

Q (curva cuadrática Bézier): La Q indica una curva cuadrática Bézier desde el punto en el que estaba el dibujo, modificándose por el "manejador" indicado por las coordenadas (x1, y1), y finalizando en las coordenadas (x, y). El "manejador" (x1, y1) actúa como un punto de control que define la dirección y longitud de la curva, creando un tramo suave entre el punto de partida y el punto final del dibujo.

V: Dibuja una línea vertical desde la posición actual del lápiz hasta un valor de coordenada y específico. Por ejemplo, V y dibujará una línea vertical hasta la coordenada y.

Puedes encontrar una descripción detallada del funcionamiento de path en https:// developer.mozilla.org/en/docs/Web/SVG/Tutorial/Paths
